IRC channel (Chat)

For occasional quick questions we have an IRC channel on Freenode: #opencast.

Technical meeting

Our weekly meeting takes place in BigBlueButton (room 1, password: welcome) on every Tuesday at 3pm UTC. Feel free to join!

Adopters meeting

Every last Wednesday of each month adopters meet at 11am EDT in the BigBlueButton (room 1, password: welcome). Feel free to join!

Bugs

Please report security related bugs by sending an email to security@opencast.org.

Everything else should be filed as a ticket in our issue tracker.

Development

Code

You can find the codebase of Opencast in the Git repository hosted at GitHub.

Pull requests

New features and bug fixes are only accepted over pull requests. There is a list of open pull requests where you also can find the official reviewer.
